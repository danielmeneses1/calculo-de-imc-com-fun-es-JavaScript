
# BMI Calculator with JS functions

System refatored of [Daniel Meneses](https://github.com/danielmeneses1).

## BMI classification table

|    BMI           |     State      |
|:----------------:|:--------------:|
|  < 18.5          |  Under weight  |
|  >= 18.5 && < 25 | Normal weight  |
|  >= 25 && < 30   | Overweight     |
|  >= 30 && < 40   | Obese          |
|  > 40            | Severe obesity |
